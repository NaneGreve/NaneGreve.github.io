Fotogrametría digital

Para este encargo se utilizó el programa Re Cap para generar una fotogrametría digital de algún alimento cualquiera. En mi caso utilicé unas naranjas para realizar el encargo.

Paso Uno:

Tuve que sacar alrededor de 50 a 100 imágenes de mi objeto. Estas imágenes al juntarse crean una malla, que se refleja en una imagen digital de mi objeto. Para conseguir esto tuve que fotografiar las naranjas desde la mayor cantidad de ángulos posibles para llegar a obtener la totalidad de este.

Paso Dos:

se traspasan las imágenes del celular o cámara fotográfica a la carpeta de fusión 360 en “default Project”.

Paso Tres:

Exportar las imágenes a Re Cap y subirlas a la nube para que luego de 8 a 24 horas el archivo esté listo para ser editado y trabajado.
