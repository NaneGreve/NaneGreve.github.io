Fusion 360 Tutoriales

Tutorial Uno

En el primer tutorial se dibujan unas líneas y generé una paralela con el comando Offset, más adelante puse Final Sketck. Luego use el comando revolve para generar una revolución en un ángulo de 360 ° grados. Finalmente se obtiene esta figura parecida a un macetero o un plato de fondo, pero lo interesantes de Fusión es que si bien ya termine de editar el sketch si no quedé conforme con mi estructura puedo editarlo en el momento y rápidamente se refrescara mi figura y quedará con las medidas que yo considere adecuadas para el encargo.

Tutorial Dos

En el segundo tutorial intente hacer el molde como el que se muestra en el video. Primero cree un Sketch y dibuje una circunferencia, luego realice una curva como la del ejemplo para finalmente aplicar el comando sweep. Para usar el comando seleccione una serie de perfiles o caras planas para definir una forma que se creará a partir de un riel o línea central.

Tutorial Tres & Cuatro

En el último tutorial aprendí a hacer la caja para los moldes, editamos la figura generando las curvas que se pueden apreciar en la imagen. Para hacer esto tuve que crear un plano de 22 x 22 mm y subdibidi mi pkano de 6 x 6, luego selecciones un par de lineas y use el comando Modfy para modificar su altura y la estructura del plano generadose una nueva geometria. Luego fui a la vista Top cree un rectangulo desde el centro de 20 x 20 mm y lo extrui en ambas direciónes, de esto se genera la caja.

Despues use el comando Thicken para dare un espesor a la fugura genera anteriormente y en las propiedades del comando le agrego la propiedad de cortar ( Cut) y se me generan el molde de dos piezas para poder prensar mi objeto.
